# Port da Home Premium → Nuxt 3 / Vue 3 (Quanta Shop)

Porte fiel do protótipo aprovado `quanta-home-premium.html` para componentes de produção,
no design system `qs-*`. Nuxt 3 SPA (`ssr:false`), Vue 3 `<script setup lang="ts">`, CSS puro
`scoped` por componente. Sem Bootstrap, sem Tailwind, sem base64.

## 3. Hospedar as fontes comerciais (Bruum FY / Kiye Sans)

Hoje os componentes declaram as famílias com fallback:
- Display (títulos): `'Bruum FY','Jost','Inter',sans-serif`
- Corpo: `'Kiye Sans','Inter','Jost',sans-serif`

Enquanto as comerciais não estão hospedadas, Jost/Inter assumem (carregue Jost+Inter no
`nuxt.config` ou via `<link>` do Google Fonts, como no protótipo).

Para ativar as fontes oficiais (sem mudar nenhum componente):
1. Coloque os `.woff2` em `public/fonts/` (ex.: `BruumFY-Bold.woff2`, `KiyeSans-Regular.woff2`, etc.).
2. Adicione um CSS global (`assets/css/fonts.css` registrado em `nuxt.config` → `css: [...]`):
   ```css
   @font-face { font-family: 'Bruum FY'; src: url('/fonts/BruumFY-Bold.woff2') format('woff2'); font-weight: 700 800; font-display: swap; }
   @font-face { font-family: 'Kiye Sans'; src: url('/fonts/KiyeSans-Regular.woff2') format('woff2'); font-weight: 400 600; font-display: swap; }
   ```
3. Assim que carregadas, as famílias já referenciadas nos componentes passam a usá-las automaticamente.

## 4. Notas de qualidade

- **Mobile-first / zero overflow horizontal** em 360/390/414px: grids colapsam, ofertas viram
  1 coluna, CEO empilha (foto em cima, película vertical, stats em linha), header vira hambúrguer.
  Alvos de toque ≥44px; tipografia fluida com `clamp()`.
- **`prefers-reduced-motion`**: hero para o autoplay e mostra o chat estático; cards sem
  transições de hover; sem animações de reveal.
- **Timers/observers** limpos em `onBeforeUnmount` no hero.
- **Tokens**: cores/raios/sombras seguem os tokens oficiais (hardcodados em hex por componente,
  como o `home-hero.vue` original já fazia, por o port ser auto-contido).
- **SEO**: `index.vue` mantém Organization + WebSite (SearchAction) + FAQPage (5) e ganhou ItemList.
  `como-funciona.vue` tem canonical próprio + HowTo.
